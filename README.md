# EDINTERN - Internship & Job Portal

A comprehensive web-based platform designed to bridge the gap between students and industry opportunities. This project features a responsive UI for browsing technical/non-technical courses and applying for internships or full-time positions.

---

## 🚀 Features

✔ **Responsive Multi-page UI**: Optimized for both Desktop and Mobile views.
✔ **Dynamic Image Slider**: Automated homepage slider for visual engagement.
✔ **Internship Registration**: Detailed form with validation for student details, domain selection, and mode (Online/Offline).
✔ **Job Application Portal**: Dedicated form for professionals to submit resumes and experience details.
✔ **Employee Login**: Secure login interface for administrative/employee access.
✔ **Accreditation Showcase**: Highlighting ISO 9001:2015 certification and IAS accreditation.

---

## 🛠️ Technologies Used

- **HTML5**: Semantic structure for all web pages.
- **CSS3**: Custom styling with Flexbox, Grid, and Media Queries for responsiveness.
- **JavaScript (Vanilla)**: Form validation, dynamic image sliding, and login logic.
- **Design**: Clean, professional UI using the 'Segoe UI' font family and a corporate blue color palette (#1f3c88).

---

## 📂 Project Structure

- `index.html` - Homepage with internship categories and course lists.
- `about.html` - Company overview, leadership team, and branch details (Salem & Chennai).
- `internshipapp.html` - Comprehensive student registration form.
- `job application.html` - Recruitment form for job seekers.
- `login.html` - Administrative/Employee login page.
- `/images` - Contains logos (ISO, IAS) and internship-related graphics.
